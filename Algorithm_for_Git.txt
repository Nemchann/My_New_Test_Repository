1. Качаешь Git с инета
2. Создаешь папку где-нибудь на компе
3. Заходишь в GitHub и создаешь удаленный репозиторий
4. Заходишь в PowerShell
5. Меняешь путь на путь в созданную папку про помощи cd
6. Пишешь git init
7. git remote add origin "Ссылка на удаленный репозиторий с GitHub без кавычек в конце .git"
8. git clone "Ссылка на удаленный репозиторий с GitHub без кавычек в конце .git" - клонируешь репозиторий
9. git status -> на ветке master (main)
10. Что-то меняешь в папке
11. git add . - добавляешь все изменения
12. git status - смотришь текущее состояние репозитория
13. git commit -m "Комментарий в кавычках" - фиксируем изменения с комментарием (делаем коммит)
14. git push origin master - запихиваем на удаленный репозиторий
15. Зайти на GitHub и все проверить

Дополнительно:
1. git checkout -b <new-branch> - создает и переключается на новую ветку
2. git checkout <branch> - переключается на существующую ветку
3. git merge <branch> - объединяет указанную ветку с текущей
4. master - главная ветка (бывает, что main)
5. Замерджить можно и на GitHub, просто тыкая кнопочки
6. git pull origin <branch> - забирает обновы из удаленного репозитория
7. git add <file> - добавляет файл (подготовка к коммиту)
